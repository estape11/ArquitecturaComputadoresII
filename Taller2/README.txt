> Para la compilacion:
		cd src
		make X
	* X puede ser pi, pi_par, saxpy, saxpy_par o IntegralesAproximacion
	
> Si se desea eliminar compilados:
		make clean

> Para ejecutar:
		./saxpy X
		./saxpy_par X Y
	* X cantidad de elementos y Y cantidad de hilos
